# Solution of task 11  

## Configure:  
autoreconf -fisv && ./configure && make  

## Usage:  
### In russian:  
LANG=ru_RU.UTF-8 ./binsearch  
### In english:  
LANG=en_EN.UTF-8 ./binsearch
### Use Roman numerals:
./binsearch -r

## Hints:  
./binsearch --version  
./binsearch --help  

## Manual:  
man -l binsearch.1
